## Getting started

First we will create a new Next JS project using the following command.

```bash
# Using npx
npx @settlemint/sdk@latest create

# Using bun
bunx @settlemint/sdk@latest create

# Using pnpm
pnpm dlx @settlemint/sdk@latest create
```
